# Lapor.in

Lapor.in merupakan aplikasi berbasis Android yang dirancang
untuk membantu masyarakat dalam menyampaikan laporan kepada
petugas desa secara terstruktur.

Aplikasi ini bertujuan untuk meningkatkan komunikasi dua arah
antara warga dan perangkat desa melalui sistem pelacakan laporan
serta penentuan prioritas aduan.

Repository ini digunakan sebagai media dokumentasi progres
pengembangan aplikasi dan pembagian tugas tim berdasarkan
metode Scrum.
